    Vorteile

    • Stabilität
    • Performance
    • Flexibilität
    • Resourcenschonend
    • Stabilität - Sieht man hier.
    • Sicherheit - Grundkonzepte der Rechteverteilung (jeder server läuft als eigener user)
    • Flexibilität - Finetuning für praktisch alles
    • Performance - weil Flexibel, kann jedes bisschen Performance herausgekitzelt werden
    • Resourcenschonend - Nur relevante Teile installieren/starten. MS Exchange für 10 Leute braucht schon 16 GB Ram
    • Shell - Irrsinning schnell für den Umgang mit dem System
